# test_githubActions

![Logo de Tecnología](https://img.icons8.com/color/96/000000/technology-items.png)  <!-- Imagen genérica de tecnología -->

Este proyecto realiza scraping automático de noticias de tecnología en España.

[The link of the page](http://jorbencasdownloaderdocument.surge.sh)

## Instalación
- Clona el repo.
- Ejecuta `pip install requests beautifulsoup4`.
- Corre el script: `python downloadFile.py`.

## Fuentes scrapeadas
- Xataka
- Genbeta
- ComputerHoy
- El Español Tech
